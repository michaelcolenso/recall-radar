# Community Signals (Typst Layout)

This directory contains a premium-style Typst layout for the provided manuscript.

## Files
- `community-signals.typ` — main document source.

## Compile
```bash
typst compile manuscript/community-signals.typ manuscript/community-signals.pdf
```

The layout uses Typst preview packages:
- `@preview/cetz` for visual canvas utilities
- `@preview/fletcher` for a ladder diagram and flow visuals
